Example - Tutorial

Menu

The menu has two drop downs which are located at the top of the application and are called 'File' and 'Options'. They both contain actions pertaning to the application.

File Dropdown

  • New

    Creates a new gear train design with a default gear set and makes left bar show the Gear Set Screen.

  • Open

    Allows the user to open an existing gear train design. (json format) Then makes left bar show GearSetScreen.

  • Save

    Allows to save the gear train currently being worked on if any. (json format)

  • Exit

    Closes the application and prompts the user if they want to save their gear train design before closing.

Top Bar

The top bar is at the top of the screen right below the menu. It has buttons for actions that can be done while a gear train design is open

Actions/Buttons

  • Auto Mesh

    If gears in a gear set are not meshing then it will bring them together and mesh them

  • Generate in Solidworks

    This will launch solidworks and create the gear train design using actual Gears, Shafts, and bearings. (Will prompt to find solidworks application executable if not found)

  • Gear Analysis

  • Shaft Analysis

  • Help

    Takes the user to this site on their default browser.

Left Bar

3D View

The 3D viewer takes up the most space on the screen. It displays the current design of the gear train with rudementary 3D models/shapes. While you edit the gear train on the left screen the 3D model will change along with it.

There are two buttons to export the current model shown in the 3D view.

Image Button

This button allows the user to save an image, in the PNG format, of the 3D view.

Model Button

This button allows the user to save a model, in the STL format, of the 3D view.
